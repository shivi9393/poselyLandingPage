# Posely iOS Landing Page

This is the Next.js landing page for the **Posely** iOS App.

## Overview
Posely is an iOS application designed to help photographers and models discover poses, save their preferred camera and lighting setups, and organize their past photo sessions.

This repository features a fully responsive, premium Next.js landing page built with React, Tailwind CSS, and Framer Motion for elegant scroll animations.

## Key Features
- **App Store Focused**: Designed exclusively for Apple App Store downloads.
- **Premium Mockups**: Framed iPhone mockups displaying actual app data inside a dynamically layered Hero and Features section.
- **Support Info**: Native Privacy Policy and Terms of Service endpoints. Contact information is globally set to `posely.circle@gmail.com`.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
